# 앱인토스 미니앱 브랜딩 가이드

앱인토스 서비스가 토스와 명확히 구분되는 브랜드 경험을 전달할 수 있도록 준비한 기준입니다.  
사용자가 토스와 앱인토스를 혼동하지 않도록 꼭 지켜주세요.

**참고**: [앱인토스 미니앱 브랜딩 가이드](https://developers-apps-in-toss.toss.im/design/miniapp-branding-guide.md)

---

## 1. 브랜드 로고 / 이름 / 컬러

### 브랜드 로고

- **노출 위치**: 전체탭, 혜택탭, 푸시, 알림, 내비게이션, 브릿지
- **적용 위치**: `granite.config.ts` → `brand.icon`에 콘솔에 업로드한 로고 **URL** 입력

**로고 제작 기준**

| 항목 | 기준 |
|------|------|
| 크기 | 600×600px **정사각형** (둥근 모서리 사용 불가) |
| 배경 | 반드시 배경 필요. 라이트/다크 모드 모두에서 잘 보이는 색상 |
| 로고에 배경 포함 시 | 600×600px 영역에 꽉 차게 배치 |

**현재 프로젝트**: `brand.icon`이 비어 있으면 테스트는 가능하나, **출시/검수 시** 콘솔 업로드 후 동일 URL을 `brand.icon`에 입력해야 합니다.

---

### 브랜드 이름

- **노출 위치**: 전체 탭, 혜택 탭, 푸시, 알림, 내비게이션, 브릿지
- **권장**: 특별한 이유 없으면 **한글** 사용 (예: `토스` O, `Toss` 비권장)
- **적용 위치**:
  - 앱인토스 콘솔에 브랜드 이름 입력
  - `granite.config.ts` → `brand.displayName`에 **동일한** 브랜드 이름 입력

**현재 프로젝트**: `brand.displayName: '캠퍼스커플'` (한글 사용 ✓)

---

### 브랜드 컬러

- **사용처**: 토스 내 진입점, 브릿지, 버튼(토스 디자인 시스템 사용 시) 등
- **선정 기준**:
  - 이미 브랜드 컬러가 있으면 그대로 사용
  - 없으면 로고에서 가장 많이 쓰인 색상 선택 (예: [컬러 추출 사이트](https://lokeshdhakar.com/projects/color-thief/))
  - 색 대비 기준 미충족 시 자동 보정될 수 있음
- **적용 위치**: `granite.config.ts` → `brand.primaryColor`에 **`#` 포함 6자리 HEX** 입력 (예: `#3182F6`)

**현재 프로젝트**: `brand.primaryColor: '#3182F6'` ✓  
앱 내 primary 색상(`app/globals.css`의 `--primary`)은 위 브랜드 컬러와 동일한 HSL(217 91% 58%)로 맞춰 두었습니다.

## 2. 내비게이션 바

화면 상단 고정 영역은 **앱인토스 전용 컴포넌트**를 사용해야 합니다.

- **게임**: [게임 가이드 - 내비게이션 바](https://developers-apps-in-toss.toss.im/checklist/app-game.md#_5-내비게이션-바)
- **비게임**: [비게임 가이드 - 내비게이션 바](https://developers-apps-in-toss.toss.im/checklist/app-nongame.md#_4-내비게이션-바)

이 프로젝트는 비게임 미니앱이므로 **비게임 가이드**를 따릅니다.

---

## 3. 탭바

- **필수 여부**: 필수 아님. 필요할 때만 사용.
- **형태**: 토스에서 제공하는 **플로팅 형태의 탭바**를 사용해 직접 구현해야 함. 자체 UI를 써도 탭바만큼은 제공 형태에 맞춤.
- **이유**: 토스 메인 하단 탭과 형태가 겹치면 사용자가 현재 위치를 헷갈릴 수 있음.

**탭바 기준**

| 항목 | 기준 |
|------|------|
| 탭 개수 | 최소 2개, 최대 5개 |
| 형태 | 토스 제공 **플로팅 형태** 유지 |

**현재 프로젝트**: `components/bottom-nav.tsx`에서 플로팅 형태로 구현됨. `(main)/layout.tsx`에서 탭바 영역에 `px-3` 좌우 여백 적용, 탭바는 `rounded-2xl`·그림자·테두리로 토스 플로팅 탭바 형태 유지. 탭 4개(메인/친구/알림/마이)로 2~5개 기준 충족.
